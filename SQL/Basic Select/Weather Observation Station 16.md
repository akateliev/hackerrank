Weather Observation Station 17
=============

Query the smallest Northern Latitude (LAT_N) from STATION that is greater than 38.7780. Round your answer to 4 decimal places.

Input Format

The STATION table is described as follows:
```
Field  Type
ID     Number
CITY   VARCHAR2 (21)
STATE  VARCHAR2 (2)
LAT_N  NUMBER
LONG_W NUMBER
```
